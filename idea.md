# AI-Medcare-Assistant — Ідея проекту

> Версія: 0.1 | Дата: 2026-03-23

---

## Концепція

**AI-Medcare-Assistant** — це персональний сімейний AI-помічник зі здоров'я.
Діє як команда: **сімейний лікар + психолог + тренер + нутриціолог + мотиватор**.

Система збирає всі дані про здоров'я кожного члена сім'ї, аналізує їх штучним інтелектом
та надає щоденні персоналізовані рекомендації — прості, зрозумілі, дієві.

---

## Цільова аудиторія

- Сім'я як базова одиниця (1–10 осіб)
- Вік: від 3 до 90 років
- Початкові користувачі: власна сім'я розробника
- Масштабування: широкий ринок (Україна → міжнародний)

## Джерела даних

### Пристрої та трекери
- Смарт-годинник (пульс, сон, кроки, SpO2, HRV)
- Телефон (кроки, активність, геолокація)
- Тонометр (тиск, пульс)
- Термометр
- Ваги (вага, відсоток жиру)
- Глюкометр

### Медичні документи
- Аналізи крові (загальний, біохімічний, розгорнутий)
- Аналіз сечі, калу
- ДНК-аналіз
- Томограми, МРТ, УЗД
- Висновки лікарів
- Виписки з лікарень
- Рецепти та призначення

### Щоденний ввід (через Telegram або додаток)
- Ранкове опитування (самопочуття, сон, настрій)
- Вечірнє опитування (що болить, енергія, стрес)
- Фото їжі (AI автоматично розпізнає, рахує калорії/БЖВ)
- Аудіо-коментарі про стан здоров'я
- Текстові нотатки

### Контекстні дані (профіль)
- Вік, стать, зріст
- Місце проживання (клімат, екологія)
- Професія та умови праці
- Образ життя (сидячий / активний / роз'їзний)
- Хронічні захворювання, алергії
- Поточні медикаменти
- Фізичні особливості (сколіоз, гіперлордоз, тощо)
- Стресові фактори

## Telegram-інтеграція

### Щоденні опитування
**Ранкове (7:00–9:00):**
- Як почуваєшся? (1–10)
- Як спав? (годин, якість)
- Чи є болі? Де?
- Настрій?
- Температура (якщо є)

**Вечірнє (20:00–22:00):**
- Що болить чи турбує?
- Рівень енергії (1–10)
- Рівень стресу (1–10)
- Що з'їв сьогодні (або фото)?
- Фізична активність?

### Швидкі дії через Telegram
- Надіслати фото їжі → AI розпізнає та записує
- Надіслати фото аналізу → AI читає та додає до картки
- Голосове повідомлення → транскрибується та аналізується
- Запитати рекомендацію → миттєва відповідь від AI

## AI-аналіз — "Команда лікарів"

### Спеціалісти в системі
| Роль | Що аналізує |
|------|------------|
| Терапевт | Загальний стан, динаміка, тривожні сигнали |
| Кардіолог | Тиск, пульс, HRV, ЕКГ |
| Гастроентеролог | Аналізи ШКТ, харчування, травлення |
| Ендокринолог | Цукор, щитовидна, гормони |
| Невролог | Сон, стрес, головні болі, втома |
| Ортопед | Постава, болі в спині, суглоби |
| Офтальмолог | Стан очей, навантаження на зір |
| Нутриціолог | Харчування, дефіцити, надлишки |
| Психолог | Стрес, вигорання, настрій, мотивація |
| Тренер | Активність, навантаження, відновлення |

### Що видає AI
1. **Щоденний бриф** — 3–5 рядків про стан сьогодні
2. **Тижневий звіт** — динаміка показників, тренди
3. **Тривожні сигнали** — якщо щось потребує уваги
4. **Рекомендації** — що робити сьогодні/цього тижня
5. **Відповіді на питання** — як з лікарем

## Харчування

### Аналіз їжі
- Фото страви → AI визначає склад
- Калорії, БЖВ, мікроелементи
- Відповідність поточному стану здоров'я
- Чи підходить для кожного члена сім'ї

### Сімейне меню на тиждень
- Генерується з урахуванням стану КОЖНОГО члена сім'ї
- Оптимізовано: максимум спільних страв (не готувати окремо)
- Де неможливо — окремі варіанти для конкретної людини
- Враховує кухаря (хто готує, скільки часу є)

### Список закупок
- Автоматично із сімейного меню
- На день / тиждень
- Групується за категоріями (овочі, м'ясо, молочне...)
- Враховує залишки (якщо підключити)

### Рецепти
- Покрокові рецепти для запланованих страв
- Адаптовані під стан здоров'я
- Прості для приготування

## Сімейний профіль

```
Сім'я "Назва"
├── Член 1: тато (вік, професія, стан)
├── Член 2: мама (вік, професія, стан)
├── Член 3: дитина 1 (вік, стан)
└── Член 4: дитина 2 (вік, стан)
```

- Кожен має свою картку здоров'я
- Спільний сімейний дашборд
- Загальні рекомендації для сім'ї
- AI враховує взаємодії (стрес тата → вплив на сім'ю)

## Дашборд

### Особистий дашборд
- Поточний стан (кольорова шкала: зелений / жовтий / червоний)
- Ключові показники великим шрифтом (тиск, вага, пульс, настрій)
- Графіки динаміки (тиждень / місяць / рік)
- Список рекомендацій на сьогодні
- Нагадування (ліки, обстеження, активність)

### Сімейний дашборд
- Стан кожного члена сім'ї одним поглядом
- Спільні завдання (меню, закупки, активності)
- Сімейні досягнення та цілі

## UX / Дизайн

### Принципи
- **Простота** — жодних складних медичних термінів без пояснень
- **Великий шрифт** — зручно для пенсіонерів
- **Кольорова інтуїція** — зелений = добре, жовтий = увага, червоний = важливо
- **Мінімум кроків** — ввести дані за 30 секунд
- **Приємний вигляд** — не медична утиліта, а дружній помічник

### Доступність
- Голосовий ввід
- Великі кнопки
- Контрастні кольори
- Проста навігація (не більше 3 рівнів)
