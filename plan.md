## 3. СТРУКТУРА БАЗИ ДАНИХ

### Ключові таблиці

```sql
-- Сім'ї
families (id, name, created_at)

-- Користувачі
users (id, family_id, name, birth_date, gender, role,
       occupation, lifestyle, location, created_at)

-- Профілі здоров'я
health_profiles (user_id, height, weight, blood_type,
                 chronic_conditions, allergies, medications,
                 physical_features, professional_risks, updated_at)

-- Щоденні опитування
daily_surveys (id, user_id, survey_type [morning/evening],
               wellbeing_score, sleep_hours, sleep_quality,
               pain_locations, mood, energy_level, stress_level,
               notes, created_at)

-- Медичні показники
health_metrics (id, user_id, metric_type, value, unit,
                recorded_at, source [manual/device/import])

-- Медичні документи
medical_documents (id, user_id, doc_type, file_url,
                   parsed_data jsonb, ai_analysis jsonb, created_at)

-- Прийоми їжі
meals (id, user_id, meal_type, photo_url,
       ai_recognition jsonb, calories, nutrients jsonb,
       eaten_at, created_at)

-- AI рекомендації
ai_recommendations (id, user_id, rec_type, content jsonb,
                    valid_until, created_at)

-- Сімейне меню
family_menus (id, family_id, week_start, menu_data jsonb,
              shopping_list jsonb, created_at)

-- Вектори для семантичного пошуку
health_embeddings (id, user_id, content, embedding vector(1536), created_at)
```

## 4. AI-АГЕНТИ — "КОМАНДА ЛІКАРІВ"

### Архітектура multi-agent системи

```python
# Кожен агент — окремий системний промпт для Claude
AGENTS = {
    "therapist":      "Сімейний терапевт — загальний стан, динаміка",
    "cardiologist":   "Кардіолог — тиск, пульс, HRV",
    "gastro":         "Гастроентеролог — ШКТ, харчування",
    "endocrinologist":"Ендокринолог — цукор, гормони",
    "neurologist":    "Невролог — сон, стрес, головний біль",
    "orthopedist":    "Ортопед — постава, суглоби, спина",
    "ophthalmologist":"Офтальмолог — зір, навантаження на очі",
    "nutritionist":   "Нутриціолог — БЖВ, дефіцити, мікроелементи",
    "psychologist":   "Психолог — настрій, стрес, вигорання",
    "trainer":        "Тренер — активність, навантаження, відновлення"
}
```

### Потік AI аналізу

```
Нові дані → Celery Task →
  → Паралельно запускаємо релевантних агентів
  → Збираємо відповіді
  → Therapist синтезує фінальний звіт
  → Зберігаємо в БД
  → Push-сповіщення користувачу
```

## 5. TELEGRAM BOT — ПОТОКИ

### Ранкове опитування (07:30)
```
Bot: "Доброго ранку, {ім'я}! 🌅 Як самопочуття?"
  → Кнопки: [1-3 Погано] [4-6 Нормально] [7-10 Чудово]
Bot: "Як спав? Скільки годин?"
  → Кнопки: [<5г] [6г] [7г] [8г] [9г+]
Bot: "Щось болить?"
  → Кнопки: [Нічого] [Голова] [Спина] [Живіт] [Інше...]
Bot: "Настрій зранку?"
  → Кнопки: [😞] [😐] [🙂] [😄]
Bot: "Дякую! Хочеш додати фото сніданку?"
  → [Так, зараз] [Пізніше] [Пропустити]
```

### Вечірнє опитування (20:00)
```
Bot: "Добрий вечір! Підведемо підсумок дня 📊"
  → Рівень енергії: [1-10]
  → Рівень стресу: [Низький / Середній / Високий]
  → Що їв сьогодні (або фото)?
  → Фізична активність?
  → Що турбує?
```

### Швидкі команди
```
/photo  — завантажити фото їжі
/scan   — завантажити аналіз (PDF/фото)
/stats  — мої показники сьогодні
/advice — рекомендація від AI прямо зараз
/menu   — меню сім'ї на тиждень
/shop   — список закупок
/family — стан сім'ї
```

## 9. ПРІОРИТЕТИ БЕЗПЕКИ (медичні дані!)
